🧠 EP/GP Loot System

View Changelog

Core System

A priority-based loot system driven by:

  • EP (Effort Points) → earned from participation
  • GP (Gear Points) → added when receiving loot
  • PR (Priority Ratio) = EP / GP

👉 Higher ratio = higher priority

How Players Gain / Lose Priority

  • Gain EP → raid attendance, boss kills, weekly quest
  • Gain GP → receiving loot
  • Weekly 10% decay reduces EP & GP
  • Base GP floor prevents abuse by new players

Why This System Exists

  • Reward consistent participation
  • Prevent loot hoarding
  • Avoid new player abuse
  • Self-correcting over time

🎯 Tiered Auction

Players choose loot priority tier:

  • High → 100% GP cost
  • Medium → 75%
  • Low → 50%
  • Off-spec → 25%

Highest priority tier wins → then highest EP/GP ratio → tie = roll

🔄 Point Decay System

  • 10% weekly decay (EP + GP)
  • 50% decay between raid phases (Base GP preserved)
  • Base GP ensures minimum fairness floor

🧪 Recipes / Patterns

  • Must be usable and learned
  • Prioritized to mains
  • BoE items go to guild bank if unusable

🎲 SR (Soft Reserve) Raids

  • Players may soft reserve items
  • SR equal for all participants
  • Priority order:
    MS (Main Spec) > OS (Off Spec) > Alts
  • +1 system applies to MS loot only

👥 Mains vs Alts

  • Alts can only roll Low or Off-spec
  • Mains always have priority over alts
  • Exception: raid leader assigned alt = treated as main

🔍 System Benefits

  • Rewards attendance & contribution
  • Prevents loot abuse & favoritism
  • Scales across multiple raid teams
  • Reduces officer workload
  • Encourages long-term engagement
